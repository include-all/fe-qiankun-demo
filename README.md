# fe-qiankun-demo
create the demo for learn qiankun;  

## start
```shell
# start main
cd main

npm install

npm run serve

# start app-vue2
cd app-vue2

npm instll

npm run serve

# start app-react
cd app-react

npm install

npm run start
```


## view
main: http://localhost:8082/

app-vue2: http://localhost:8083/subapp/app-vue2/

app-react: http://localhost:8084/subapp/app-react/
